## 🚀 GitHub Pages Deployment

1. Create a new repo: `digitalsevacenter` on GitHub
2. Upload all files maintaining folder structure
3. Go to **Settings → Pages**
4. Set source: **Deploy from branch → main → / (root)**
5. Your site will be live at: `https://yourusername.github.io/digitalsevacenter`

### Custom Domain (digitalsevacenter.in)
1. Add a file named `CNAME` with content: `digitalsevacenter.in`
2. In your domain registrar, add DNS records:
   - A record: `185.199.108.153`
   - A record: `185.199.109.153`
   - A record: `185.199.110.153`
   - A record: `185.199.111.153`
3. GitHub Pages → Custom domain → Enter `digitalsevacenter.in`
