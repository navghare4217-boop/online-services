GST Return Filing: GSTR-1, GSTR-3B, GSTR-9 कब और कैसे?

GST Registration लेने के बाद regular return filing mandatory है। Late filing पर penalty लगती है इसलिए deadlines समझना जरूरी है।

GST Returns & Due Dates

  • GSTR-1: Outward supplies – 11th of next month (monthly) या quarterly
  • GSTR-3B: Summary return + tax payment – 20th of next month
  • GSTR-9: Annual return – 31 December
  • GSTR-4: Composition dealers – Annual, 30 April

Late Fee

GSTR-3B: ₹50/day (₹20 for nil return) | Maximum ₹10,000 per return

Tips to Avoid Mistakes

  • All invoices correctly enter करें
  • Reconcile with purchase invoices
  • ITC claim correctly करें
  • Nil return भी file करें
