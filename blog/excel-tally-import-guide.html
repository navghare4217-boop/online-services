Excel Data को Tally में Import कैसे करें? Step-by-Step

Tally ERP में manually data enter करना time-consuming है। Excel से Tally में data import करने से hours का काम minutes में होता है।

Excel to Tally Import के Methods

  • TDL Script: Tally Definition Language से custom import
  • Excel to Tally Utility: Third-party software use करना
  • Tally XML Import: Data को XML format में convert करना

क्या Import हो सकता है?

  • Ledger Masters (Parties, Accounts)
  • Sales & Purchase Vouchers
  • Stock Items & Groups
  • Opening Balances
  • GST data

Common Issues & Solutions

Format mismatch, duplicate entries, GST code errors जैसी problems आती हैं। Expert से help लेना बेहतर है।

Excel to Tally Import क्यों ज़रूरी है?

अगर आपका data पहले से Excel में है — Sales, Purchase, Stock, Ledgers — तो उसे manually Tally में enter करना बहुत time-consuming है। Excel to Tally Import से हज़ारों entries मिनटों में import हो जाती हैं।

Tally में Import के Methods

MethodSuitable ForDifficulty
XML Import (Built-in)Tally ERP/Prime usersMedium
TDL (Tally Definition Language)Custom formatsAdvanced
Third-party tools (TallyConnect, etc.)Large dataEasy
ODBC/APIDevelopersExpert

XML Method — Step by Step

Step 1 — Excel Data तैयार करें: Excel में columns होनी चाहिए: Date, Voucher No, Ledger Name, Amount, DR/CR। Date format: DD-MMM-YYYY (01-Jan-2025)

Step 2 — XML File बनाएं: Excel template से XML generate करें। Tally XML structure follow करना होता है।

Step 3 — Tally में Import करें: Tally Prime खोलें → Gateway of Tally → Import Data → Vouchers। XML file select करें। Import होगा।

Common Errors और Solutions

  • "Ledger not found": Tally में पहले ledger create करें
  • Date format error: Excel में date format check करें — DD-MMM-YYYY होना चाहिए
  • "Invalid XML": XML validator से check करें
  • Duplicate entries: Voucher numbers unique होने चाहिए

Excel to Tally Import Tools

  • TallyConnect: Popular third-party tool, easy to use
  • Excel2Tally: Free basic version available
  • Gyan Data Importing Tool: Used by CAs widely
